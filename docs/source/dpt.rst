Package DPT
===========

Base du jeu
+++++++++++

Le package ``dpt`` contient l'integralité des mécanismes de base du jeu. Il définit la structure de base avec :py:mod:`dpt.game.Game`. L'exécutable permet simplement d'intégrer un interpréteur python compatible avec plusieurs platformes. Cela évite l'installation de python pour les joueurs classiques. Le package ``dpt`` n'a pas été conçu pour être modifiable contrairement aux :doc:`ressources`.

Participer au développement
+++++++++++++++++++++++++++

Nous invitons toutes les personnes voulant participer au développement du jeu à nous contacter sur discord (Dwight Studio Hub https://discord.gg/yZwuNqN). Ce n'est pas obligatoire mais simplement appréciable. 

Pour apporter des modifications au jeu, vous pouvez effectuer des Pull-Request sur le Github du projet. Nous allons valider ou non vos modifications et les intégrer au jeu ou non. Vous serez évidemment crédité.
